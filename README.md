# Conway's Game of Life Implemented in PyGame
A simple implementation of Conway's Game of Life.
Works much much better than the tkinter version I made.
No interactions built in yet.
## Requirements
```
pip install pygame
```
## Usage
All settings are set by editing the program.
```
conway.py
```
